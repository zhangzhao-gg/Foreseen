# 先见 Foreseen

> 这不是一个帮你做决定的工具。
> 这是一面镜子——但镜子里的像，比你更早动。

![screenshot](screen.png)

## 它是什么

**先见**是一个交互式心理体验装置。你给它一个没想清楚的决定，它陪你思考、给你选项、观察你的犹豫——然后在结束时，给你一份比你自己更了解你的档案。

整个体验发生在一张古老的书页上。没有导航，没有页面跳转。墨水渗入，墨水褪去，像一本预言之书在回应你写下的每一个字。

**灵感来源：** 心跳文学部 · 认知心理学 · 控制感剥夺实验

## 核心体验

- **书页隐喻** — 界面是一本书，不是一个 App。羊皮纸底色，衬线字体，羽毛笔光标
- **墨水动效** — 用户文字褪去，系统文字从纸背面渗出，选项从书页边缘生长
- **行为观察** — 系统默默记录你的犹豫、停顿、来回——然后在对话中轻描淡写地提起
- **结束洞察** — 对话结束时，AI 生成一份关于你的深度洞察，时间戳精确到秒
- **幽灵打字** — AI 偶尔预言你的下一句话，逐字渗入输入框，你打字即夺回控制权

## 技术栈

```
React 19 + TypeScript + Vite 8 + GSAP
MiniMax API（对话引擎，可选）
纯前端 SPA，无后端
```

## 快速开始

```bash
# 克隆
git clone https://github.com/your-username/foreseen.git
cd foreseen/foreseen-app

# 安装依赖
npm install

# （可选）配置 AI 对话
cp .env.example .env
# 编辑 .env，填入 MiniMax API Key
# 不配置也能用——回退预设文案，体验完整性不依赖外部服务

# 启动开发服务器
npm run dev
```

## 项目结构

├── foreseen-app/          # 主应用
│   ├── src/

## 设计哲学

**书不是在回应你，书是在显现它本来就知道的东西。**

这个隐喻解决了一个根本问题——为什么用户会相信系统比他们更了解自己。墨水消失是正常的，文字自己出现是正常的。在预言之书的世界里，这恰恰是它工作的方式。

### 视觉系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 书页底色 | `#F5EDD6` | 泛黄羊皮纸 |
| 用户墨水 | `#1C1208` | 你写下的字 |
| 系统墨水 | `#0F0A04` | 从纸背面渗出的字 |
| 褪色墨水 | `#9C8E78` | 消退中的水印 |
| 边框线条 | `#8B7355` | 手划的批注框 |
| 页面之外 | `#1A1410` | 烛光照不到的黑暗 |

### 字体

- **正文** — Cormorant Garamond（衬线，有分量的思考）
- **时间戳** — Courier Prime（等宽，打字机质感）

## 构建与部署

```bash
cd foreseen-app
npm run build    # 输出到 dist/
```

产物是纯静态文件，任何静态托管服务均可部署（Nginx、Vercel、Cloudflare Pages 等）。
